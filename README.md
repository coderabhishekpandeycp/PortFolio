# 🚀 Abhishek Pandey - MERN Stack Developer Portfolio

A modern, sleek portfolio website built with **React**, **Node.js**, **Express**, and **MongoDB**. Showcasing projects, skills, and experience in full-stack web development.

## 🌟 Features

- ✨ Responsive and modern UI design
- 🎨 Dark/Light theme with glassmorphism effects
- 📱 Mobile-first approach
- 🚀 Smooth animations and transitions
- 📊 Projects showcase with live demos
- 💬 Contact form with email notifications
- 🔍 SEO optimized
- ⚡ Fast performance

## 🛠 Tech Stack

### Frontend
- React.js with Hooks
- Tailwind CSS for styling
- Framer Motion for animations
- Axios for API calls
- React Router for navigation

### Backend
- Node.js & Express.js
- MongoDB with Mongoose
- JWT for authentication
- Nodemailer for emails
- CORS enabled

## 🚀 Getting Started

### Prerequisites
- Node.js (v14+)
- MongoDB
- Git

### Installation

1. Clone the repository
```bash
git clone https://github.com/coderabhishekpandeycp/PortFolio.git
cd PortFolio
```

2. **Backend Setup**
```bash
cd backend
npm install
cp .env.example .env
# Edit .env with your values
npm start
```

3. **Frontend Setup**
```bash
cd ../frontend
npm install
npm start
```

Visit `http://localhost:3000` 🎉
